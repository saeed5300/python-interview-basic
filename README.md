Python Interview Basics for Beginners (Self-Created Project)

This repository is a self-created educational project designed to cover fundamental Python concepts commonly asked in beginner-level technical interviews.

 Topics Covered
Variables and data types (int, float, string, boolean)
String operations and formatting
Arithmetic, comparison, and logical operators
Conditional statements (if / else)
Basic input/output using print()
💡 Example Code
name = 'sara"
age = 22

print("Name:", name)
print("Age:", age)

if age > 18:
    print("Adult")
else:
    print("Minor")
Sample Interview Questions
What is a variable in Python?
Difference between = and ==
What are Python data types?
How do you concatenate strings?
 Learning Outcome

This project helps beginners understand core Python concepts and prepare for entry-level programming interviews.
